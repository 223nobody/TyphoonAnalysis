## 2. 时序图

### 2.1 台风预测流程

```plantuml
@startuml
participant 用户
participant 前端
participant PredictionAPI
participant TyphoonPredictor
participant LSTMModel
participant Database
用户 -> 前端: 选择台风和预测参数
前端 -> PredictionAPI: POST /api/predictions/predict
PredictionAPI -> Database: 查询台风历史路径
Database --> PredictionAPI: 返回路径数据
PredictionAPI -> TyphoonPredictor: 调用预测方法
TyphoonPredictor -> TyphoonPredictor: 数据预处理
TyphoonPredictor -> LSTMModel: 模型推理
LSTMModel --> TyphoonPredictor: 预测结果
TyphoonPredictor --> PredictionAPI: 返回预测路径
PredictionAPI -> Database: 保存预测记录
PredictionAPI --> 前端: 返回预测数据
前端 --> 用户: 显示预测路径
@enduml
```

### 2.2 AI客服对话流程

```plantuml
@startuml
participant 用户
participant AIAgentUI
participant ASRAPI
participant AIAgentAPI
participant QuestionDB
participant AIService
用户 -> AIAgentUI: 输入问题(文字/语音)
AIAgentUI -> ASRAPI: 语音识别(可选)
ASRAPI --> AIAgentUI: 返回文字
AIAgentUI -> AIAgentAPI: POST /api/ai-agent/chat
AIAgentAPI -> QuestionDB: 匹配预设问题
QuestionDB --> AIAgentAPI: 匹配结果
AIAgentAPI -> AIAgentAPI: 未命中预设
AIAgentAPI -> AIService: 调用AI模型
AIService --> AIAgentAPI: 流式响应(SSE)
AIAgentAPI --> AIAgentUI: 流式返回答案
AIAgentUI --> 用户: 逐字显示回答
@enduml
```

### 2.3 数据爬取流程

```plantuml
@startuml
participant Scheduler
participant CMACrawler
participant ActiveTyphoonCrawler
participant 中国气象局API
participant Database
Scheduler -> CMACrawler: 触发定时任务(每10分钟)
CMACrawler -> 中国气象局API: 请求台风列表
中国气象局API --> CMACrawler: 返回JSON数据
CMACrawler -> CMACrawler: 解析数据
CMACrawler -> Database: 更新台风基本信息
Scheduler -> ActiveTyphoonCrawler: 触发活跃台风爬取
ActiveTyphoonCrawler -> 中国气象局API: 请求台风详情
中国气象局API --> ActiveTyphoonCrawler: 返回路径数据
ActiveTyphoonCrawler -> 中国气象局API: 请求多机构预报
中国气象局API --> ActiveTyphoonCrawler: 返回预报数据
ActiveTyphoonCrawler -> Database: 更新路径和预报
@enduml
```

## 5. 活动图/流程图

### 5.1 LSTM路径预测流程

```plantuml
@startuml
start
:接收预测请求;
:查询台风历史路径;
if (路径点数量>=12?) then (是)
:提取最近12个路径点;
:特征工程处理;
:数据标准化;
:LSTM编码器处理;
:注意力机制加权;
:多头预测输出;
:反标准化处理;
:计算置信度;
:保存预测记录;
:返回预测结果;
else (否)
:使用线性外推降级方案;
:返回降级预测结果;
endif
stop
@enduml
```

### 5.2 预警生成流程

```plantuml
@startuml
start
:获取活跃台风数据;
:遍历每个台风;
repeat
:获取最新位置;
:计算与监测点距离;
:获取最大风速;
:获取中心气压;
if (风速>=51m/s?) then (是)
:设置红色预警;
elseif (风速>=41m/s?) then (是)
:设置橙色预警;
elseif (风速>=33m/s?) then (是)
:设置黄色预警;
elseif (风速>=17m/s?) then (是)
:设置蓝色预警;
else (否)
:无预警;
endif
repeat while (还有台风?) is (是)
:汇总预警信息;
:推送到前端;
stop
@enduml
```

### 5.3 用户认证流程

```plantuml
@startuml
start
:用户提交登录请求;
:验证用户名密码;
if (验证通过?) then (是)
:生成JWT Token;
:设置Token过期时间;
:返回Token给前端;
:前端存储Token;
:后续请求携带Token;
else (否)
:返回认证失败;
:提示用户重试;
endif
stop
@enduml
```

## 使用说明

1. 复制对应的 PlantUML 代码块（不含 ``` 标记）
2. 在飞书文档中插入画板
3. 点击「PlantUML」选项
4. 粘贴代码即可渲染

注意：所有代码均遵循飞书画板的安全子集语法，无行首缩进，无样式指令。
